# vue 创建 a 标签下载图片文件




## 谷歌浏览器

```
let aLink = document.createElement('a')
aLink.download = 'name.png' ;
aLink.href = $('#qrcode img').src; 
// aLink.click(); // 支持谷歌，火狐暂不可用
aLink.dispatchEvent(new MouseEvent('click', {})); // 谷歌/火狐都可用
```
