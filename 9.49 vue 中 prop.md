# prop


组件中传递 使用 prop

```
export default {
    name: 'vue-gic-card',
    props: {
        // 检测类型
        height: Number,
        // 检测类型 + 其他验证
        age: {
          type: Number,
          default: 0,
          required: true,
          validator: function (value) {
            return value >= 0
          }
        }
```
